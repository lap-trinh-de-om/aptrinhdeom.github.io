[Phần 4] Vim là cái quái gì? - Làm đẹp Vim với Vim Airline

Làm đẹp thêm phần ch0n phía dưới với Vim Airline, thêm vài thông tin để biết file name, git branch, file type...

Jun 12, 2021
2 phút để đọc

Theme giao diện: [Vim Airline](https://github.com/vim-airline/vim-airline)

Vim airline là plugin để làm cho cửa sổ terminal xài Vim của bạn không bị nhàm chán, nó có những thứ phía dưới như sau:

Khá đẹp đúng không, cho mình biết được những thông tin như:

  • NORMAL: mode của vim hiện tại, có thể là NORMAL, INSERT, VIRSUAL
  • Git branch là gì
  • Tên file hiện tại
  • Ngôn ngữ của file đó là gì
  • Encode của file là gì: utf-8
  • Phần trăm của con trỏ so với tổng số dòng
  • Số dòng hiện tại / tổng số dòng của file (ln:15/65)
  • Cột của con trỏ đang đứng (col:1)

Rồi giờ cài đặt và cấu hình nó nào:

Cài đặt Vim Airline:

vim ~/.vimrc

call plug#begin()
...
Plug 'vim-airline/vim-airline'
Plug 'vim-airline/vim-airline-themes'
call plug#end()

Gõ :w để lưu, xong gõ tiếp:

:source ~/.vimrc
:PlugInstall

Cài thêm font power-airline vô nữa thì nó mới hiển thị mấy cái symbol (ký tự) của branch, mũi tên đúng được:

Trên ubuntu hoặc debian hoặc các Linux distro dựa trên debian:

sudo apt install fonts-powerline

Trên MacOS hoặc các OS khác:

# clone
git clone https://github.com/powerline/fonts.git --depth=1
# install
cd fonts
./install.sh
# clean-up a bit
cd ..
rm -rf fonts

=> Đổi font của terminal lại thành "Source Code Pro for Powerline"

Vậy là cài xong, giờ cấu hình nào.

Cấu hình Vim Airline:

Các bạn vào link này để xem các theme hiện có của Airline nhé: https://github.com/vim-airline/vim-airline/wiki/Screenshots

Mình lựa được theme bubblegum thấy dẹp đẹp, nên quăng vô xài thôi 😍

let g:airline_theme='bubblegum'
let g:airline_powerline_fonts=1

Cấu hình cho các symbol hiển thị đúng:

if !exists('g:airline_symbols')
  let g:airline_symbols = {}
endif
  
"" powerline symbols
let g:airline_left_sep = ''
let g:airline_left_alt_sep = ''
let g:airline_right_sep = ''
let g:airline_right_alt_sep = ''
let g:airline_symbols.branch = ''
let g:airline_symbols.readonly = ''
let g:airline_symbols.linenr = 'ln'
let g:airline_symbols.maxlinenr = ''
let g:airline_symbols.dirty='⚡'
let g:airline_symbols.colnr='col'"

Cấu hình để hiển thị branch name:

let g:airline#extensions#tabline#enabled = 1
let g:airline#extensions#hunks#enabled=0
let g:airline#extensions#hunks#coc_git = 1

Vậy là xong, tận hưởng thành quả nào.
